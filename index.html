ℹ️ Sistema de Puntuación

PREDICCIONES DE PARTIDOS

🎯
3 puntos — Resultado exacto
Predijiste el marcador exacto. Ej: predijiste 2-1 y el partido terminó 2-1.
✅
1 punto — Tendencia correcta
Acertaste quién gana o que hay empate, pero no el marcador exacto. Ej: predijiste 1-0 y fue 3-0.
❌
0 puntos — Resultado incorrecto
No acertaste ni el resultado ni el marcador. Ej: predijiste que ganaba el equipo 1 y ganó el equipo 2.

PREDICCIONES ESPECIALES

🥇 Campeón · 🥈 Subcampeón · 🥉 Tercer puesto · ⚽ Goleador
Cada predicción especial correcta otorga 5 puntos adicionales al final del torneo. ⏳ Plazo para completarlas: viernes 19 de junio.

EJEMPLOS

🇧🇷 2 - 0 🇦🇷 (resultado real)
→ Predijiste 2-0 → +3 pts
→ Predijiste 1-0 → +1 pt
→ Predijiste 0-1 → 0 pts
