# ⚽ Polla Mundial ProntoPaga 2026

Aplicación web de predicciones para el Mundial FIFA 2026.

---

## 📋 Funcionalidades

| Sección | Descripción |
|---|---|
| 📝 Predicciones | Predice los marcadores de la fase de grupos |
| 📋 Mis Picks | Revisa tus predicciones guardadas |
| 🌍 Grupos | Tablas de posiciones y calendario |
| 🏆 Tabla | Ranking de participantes |
| 📊 Estadísticas | Gráficos de puntos y exactitud |

---

## 🏆 Sistema de puntos

- **3 puntos** — Marcador exacto
- **1 punto** — Tendencia correcta (ganador o empate)
- **0 puntos** — Resultado incorrecto
- **5 puntos** — Cada predicción especial correcta (campeón, subcampeón, tercer puesto, goleador)
